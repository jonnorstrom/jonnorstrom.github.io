CSS Concepts

~ February 7th ~

What is the difference between margin, border, and padding?

I'd say the answer is easier than it seems at first. Let's start from the inside out. First you have your content. Then you have your padding. This is the distance between your content and your border. Then you have your border. This marks the end of this specific element. Simple. Then the margin is the distance between this element and the next.

What are the best practices using classes and id's?

Keep classes more general or broad. Id's are only supposed to be used to identify one single item.

What are the diffences between relative, absolute, static, and fixed positioning?

Relative is the position relative to where it would normally sit. Static positioning is the default positioning. Fixed positioning positions the element to the browser window. And absolute positioning positions the element to it's first positioned ancestor.

What does it mean to display inline vs. inline-block?

Inline objects allow HTML to sit next to it, but don't allow for width or height settings, while inline-block allows for those but it treats the elements like a block.
