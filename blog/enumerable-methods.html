Enumerable Methods

~ February 17th ~

What are enumerables?

Well there are quite a few definitions floating throughout the web, but David A. Black points out nicely in his Well Grounded Rubyist: 2nd Ed. that Enumberable is a module in which it can be instanced through. A great way of instancing through is with the .each, in which the following block of code will be run for each object in the container for which you've called .each.

While there are many enumerable methods to help keep your code cleaner and shorter, one that I think is pretty important, and equally confunsing. With this method, you can sum together all the numbers from an array, make a new hash (with a key => value object to your specifications, and a handsome slew of other things. But the main thing here is that you can pass a block of clode on to each item in your container - with a starting value, whereas the .each enumerator doesn't quite let you do just that.
